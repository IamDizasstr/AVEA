
## Особенности/Фичи:
* Поддержка разных языков: Basque, Belarusian, Czech, English, French, German, Japanese, Spanish, Polish, Russian
* Текущая погода
* Прогноз на 7 дней
* Множество локаций
* Настройки
* Без рекламы
